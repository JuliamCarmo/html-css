Para adicionarmos um vídeo em HTML5, precisamos das seguintes tags:

        
          <video width="500" poster="imagens/imagem01.jpg" controls> 
          <source src="midia/Video-m4v-5823521.m4v" type="video/mp4"> 
          <source src="midia/Video-webm-5823521.webm" type="video/webm">
          <source src="midia/video-5823521.mp4" type="video/mp4">

É importante usarmos formas diferentes de reprodução de vídeos para diferentes tipos de navegadores conseguirem assistir ao vídeo (m4v, webm, mp4 e ogv são os mais utilizados).

Inserindo vídeos do YouTube

Se liga nesse trailer de Halo 5

Este vídeo foi buscado do canal do YouTube. Para isso precisamos seguir os seguintes passos Buscar pelo vídeo desejado (conferir os direitos autorais) -> Clicar na opção compartilhar -> Incorporar -> Copiar (ctrl C) e colar na pagina de programação (ctrl V). A nossa maior vantagem em utilizar este método, é não estarmos utilizando dados do nosso servidor de hospedagem, porem, o Youtube irá mostrar seus anuncios na nossa página atraves do vídeo que adicionamos.
