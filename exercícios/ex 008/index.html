Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negito usando a tag < b > (não semantica)
Nesta frase temos um termo em destaque usando a tag < strong > (semantica)

Itálico / Enfase

Nesta frase temos um termo em itálico usando a tag < i > (não semantica)
Nesta frase temos um termo em ênfase usando a tag < em > (semantica)
(é recomendado que se utilize preferencialmente as tags < strong > quando necessario usar negrito e a tag < em > quando necessario usar italico)

Texto marcado

Podemos criar também um texto marcado utilizando a tag <mark>.
(Para mudar a cor da marcação, devemos utilizar a CSS style dentra da tag de abretura <mark>, e então escolher a opção background-color)

(OBS: Caso esteja escrevendo um texto que exitem muitos trechos grifados, o ideal e mais prático, é que você crie abaixo da tag <title> uma tag <style> e dentro da tag style crie a tag <mark> e então escolha a opção background-color e defina uma cor para grifar todas as palavras ou frases marcadas.
Ex: "Meu conhecimento é a minha distração, coisas que eu sei...medo mora perto das ideias loucas".

Textos grandes e pequenos

Estamos criando um texto grande e um texto pequeno neste paragrafo.
Para criar frases com letras um pouco maiores, basta utilizar a tag <big> (Contudo, é importante resaltar que a tag <big> já esta obsoleta de acordo com a W3C).
Para criar frases um pouco menores, basta utilizar a tag <small> (De acotrdo com a W3C, essa tag ainda não esta obsoleta).

Texto Deletado

Podemos marcarum texto como excluído para indicar que ele deve ser lido, mas não considerado. (Para isso utilizamos a tag <del>)

Podemos marcar um texto como incerido para dar ênfase e indicar que ele foi adicionado depois. (Para isso utilizamos a tag <ins>, lembrando que a tag <ins> é uma tag semântica)
Existe também o sublinho com a tag <u> (não semântica).

Texto Sobrescrito

Para inserir coisas do tipo x20+3. (Neste caso utilizamos a tag <sup>)

Texto Subscrito

Para inserir coisas do tipo H2O. (Neste caso utilizamos a tag <sub>)
