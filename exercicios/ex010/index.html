(Os links externos são criados com a tag <a> e dentro da tag <a> adicionamos href="" para encaminharmos o link.)

É muito importante sempre lembramos também de colocarmos dentro da tag de abertura <a> target="_blank" rel="external". Isso permite que a nossa página ainda fique aberta quando os links forem acessados.

Usando Links Internos

Para mais informações sobre Links Internos, você pode acessar a minha segunda pagina clicando aqui. (Neste caso, ao clicar no link uma nova aba sera aberta no navegador. Isso foi feito com as tags <a href="pag002.html" target="_blank" rel="external" >

(Também podemos ir para a proxima pagina clicando neste link sem abrir uma nova aba no navegador.) Para isso utilizamos as tags <a href="pag002.html" rel="next">
