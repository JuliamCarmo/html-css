Esta é a segunda pagina do meu site

Para vir ate esta pagina, é importante lembrar que todos os arquivos estão dentro do mesmo documentos nos meus arquivos (ex 010). As tags utilizadas são <a href="pag002.html" target="_blank" rel="external">. Essas tags nos direcionou para essa pagina e permitiu que a outra permanecesse em aberto.

Porem caso o site tenha muitas paginas para direcionar, não seria interessante usar a tag <target="_blank" rel="external">, porque muitas abas ficariam abretas no navegador do usuário.

Para que isso não aconteça, podemos criar novos links dentro das novas paginas que nos faça voltar a pagina anterior usando as tags < a href="../titulo da pagina.html"> para voltarmos a pagina inicial. E também a tag <rel="next"> para irmos para a proxima pagina, assim como tambem a tag <rel="prev"> que nos leva a pagina anterior.

Por exemplo: Esta pagina esta sendo patrocinada pela Amazon Prime Vídeo.(Aqui usamos as tags <a href="https://www.amazon.com.br/" target="_blank" rel="nofollow"> com o link da pagina da Amazon)
