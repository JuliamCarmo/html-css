Imagem Dinamica

Tente abrir este site em vários dispositivos diferentes, ou simplesmete aumente e diminua o tamanho so seu navegador.

Bora ver a parte tecnica dessa bagaceira

Apesar de ser trabalhoso precisar criar diferentes tamanhos de imagens, é gostosinho ver o site se adaptando e ficando bonitnho quando aumentamos e diminuimos a pagina.

Quando estivermos organizando a sequencia das fotos, é muito importante que seja organizado de baixo para cima, iniciando sempre pela imagem grande! As tags que utilizamos são:

        
            < source media="(max-width:750px)" srcset="imagens/imagem.p.png" type="image/png" >
            < source media="(max-width:1050px)" srcset="imagens/imagem.m.png" type="image/png" >
            < img src= "imagens/imagem.g.png" alt= "imagemzinha grande" >

Reproduzindo Audio

Vamos apender a reproduzir audio em HTML5

Temos também outras opções de adicionar musiquinhas

Bora ver a parte tecnica bro

Para adicionarmos um audio a nossa pagina, antes de qualquer coisa, precisamos nos lembrar dos direitos autorais antes de baixa-lo e usarmos.
As tags que utilizamos são as:
<audio src="midia/8bit dreamscape the whole other.mp3" controls autoplay>

Também temos outras opções de formato de audio, que seria alem do MP3, o OGG e a WAV (a WAV infelizmente é um formato muito pesado, por isso não é muit recomendada). Esses outros dois formatos serão acionados caso o navegador do usuario não consiga reprodizir o audio primeiro em MP3. Fazemos isso usando as seguintes tags:

        
            <source src="midia/all the fixings zachariah hickman.mp3"  type="audio/mpeg">
            <source src="midia/all the fixings zachariah hickman.ogg"  type="audio/ogg">

Caso ainda assim nenhuma das tres opções funcione no navegador do usuario, podemos adicionar a opção para fazer download do arquivo. Fazemos isso utilizando as seguintes tags:
<a href= "midia/all the fixings zachariah hickman.mp3">Clic aqui para baixar o arquivo MP3
