Paragrafos e quebras de linhas

Na linguagem HTML podemos escrever um poema sem precisar criar um novo paragrafo para cada frase. Basta utilizarmos o <br> para isso. Por exemplo:
Amor é fogo que arde sem se ver,
É ferida que dói e não se sente,
É um contentamento, descontente
É dor que se desatina sem doer. 💕 🔥
