Adicionamos a tag <ol> para iniciar uma Lista Ordenada, e a tag <li> para adicionar os componentes da lista. Geralmente a lista é ordenada por sequenvcias numericas (item 1, 2, 3...) mas caso queira mudar, podemos usar dentro da tag de abertura <ol> type="" ou caso queira mudar o inicio da númeração, podemos usar start="número desejado"
